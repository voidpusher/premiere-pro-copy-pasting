Instant Paste is installed

This is a Premiere Pro extension, so it runs inside Premiere instead of as a separate Windows app.

  1. Fully close and reopen Adobe Premiere Pro.
  2. Open any Premiere project.
  3. Go to Window > Extensions > Instant Paste.
  4. Sign in with the email you used at purchase.

You can search Instant Paste in the Windows Start Menu anytime to reopen these instructions.
